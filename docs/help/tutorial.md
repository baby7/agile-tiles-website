# 教程

## 一、如何进入卡片设计界面？

1. 可以在顶部的【更多】按钮处打开卡片设计界面

2. 可以在【设置】➡️【卡片设计】处打开卡片设计界面

<img src="https://index.agiletiles.com/website/images/help/enter-the-card-design.png" alt="进入卡片设计" style="display: block; margin: 0 auto; width: 40%;" />

## 二、如何设计和添加卡片

* 左侧为卡片商店，可以在这里添加卡片

* 右侧为卡片设计模块，右侧上方是操作按钮，可以扩展布局宽度、删除卡片

* 右侧为卡片设计模块，右侧下方是卡片设计器，可以拖动、置换卡片

* ❗【重要】最后一定要点击右侧上方的【确定】按钮来保存布局
